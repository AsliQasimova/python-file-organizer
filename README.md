# python-file-organizer

This Python script automatically organizes files into folders based on file type.

## Features
- Organizes images, documents, music, videos, and archives
- Automatically creates folders if they don't exist
- Easy to customize file types and destination paths
- Uses only built-in Python libraries (`os`, `shutil`)

## How to Use
1. Clone or download this repository.
2. Open `File_Organizer.py` in your editor.
3. Set your **source folder** and **destination folder** in the script:
   ```python
   SOURCE = "your/source/folder/path"
   DESTINATION = "your/destination/folder/path"
